Past OS/2-only releases

These were had more of a source-available license than open source, though source code for the final release was never published. Since the closure of the Kermit project, these should now be under the Revised 3-Clause BSD license despite what any included licensing materials may say.

This list is not comprehensive. During much of this time, the OS/2 code was just part of the regular C-Kermit distribution and may or may not have been updated in any given C-Kermit release. C-Kermit 5A(189) appears to have been the first OS/2-specific release.

  • OS/2 C-Kermit 5A(191) - 23 April 1995. Last free release for OS/2 prior to Kermit 95 3.0.0 Beta 7.
    • Binaries
  • OS/2 C-Kermit 5A(190) - 4 October 1994. Last release to include source code or 16bit OS/2 support.
    • Binaries
    • Source code
  • OS/2 C-Kermit 5A(189) - 18 July 1993
  • OS/2 C-Kermit 5A(188) - 23 November 1992, updated 23 November 1992
  • OS/2 C-Kermit 5A(179) - 14 February 1992
  • OS/2 C-Kermit 4F(091) (OS/2 Kermit 1.00)
  • OS/2 C-Kermit 4E(070) (OS/2 Kermit 1.00) - 10 May 1989
  • OS/2 C-Kermit 4E(070) (OS/2 beta test version 1.0p) - 15 March 1989
    • Binaries (encoded - see ckboo.txt for details)
  • OS/2 C-Kermit 1.0b - 5 August 1988
  • OS/2 C-Kermit 1.0a - 29 July 1988
