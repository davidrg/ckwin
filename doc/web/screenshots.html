Kermit 95 Screenshots

Kermit 95 supports virtually all 32bit and 64bit versions of Microsoft Windows regardless of CPU architecture, along with 32bit IBM OS/2. This page shows screenshots of Kermit 95 running on these platforms. For older screenshots, see The Kermit Project Website.

By Platform

  • Windows NT (incl. 2000/XP/Vista/7/8/etc)
  • Windows 95/98/ME
  • IBM OS/2
