Kermit 95 Documentation

Three versions of the Kermit 95 manual are currently available online in addition to being packaged with Kermit 95:

  • Kermit 95 v3.0 - latest stable release (3.0.0 beta 7 currently)
  • Kermit 95 latest development build - may discuss features and details that are not available yet
  • Kermit 95 v2.1.3 - the last commercial release from 2003

To download Kermit 95, the latest stable release is: Kermit 95 3.0 beta 7. Open Source, no registration required. Works on:

  • all 32-bit and 64-bit desktop versions of Windows on any CPU architecture, and most server versions (excludes Windows NT 3.10)
  • IBM OS/2 2.x, Warp 3, Warp 4

For support, you can ask a question on Github Discussions.

For information on the control sequences supported by Kermit 95, see the draft Kermit 95 Control Sequences document. Note that this document covers the next release of K95, not the current one! If you need to know what control sequences are supported in beta 7, you can find a copy of this file in your DOCS folder.
